# hadoop docker 环境配置

使用 docker 搭建 hadoop 环境

## conf

`etc/hadoop` 下的一些配置文件

```bash
conf
├── core-site.xml
├── hadoop-env.sh
├── hdfs-site.xml
├── mapred-site.xml
├── workers
└── yarn-site.xml
```

## ssh 登录的 key

`ssh` 免密登录

```bash
ssh
├── authorized_keys
├── id_rsa
└── id_rsa.pub
```
